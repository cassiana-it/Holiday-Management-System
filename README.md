# Holiday Management System (Java)

A console-based Java application for managing holiday bookings.
This project demonstrates the use of object-oriented programming, file handling, and data persistence.

## Features

* Add a new holiday
* View all holidays
* Filter holidays by price range
* Display the cheapest holiday
* Update holiday details
* Remove holidays
* Persistent data storage using file serialization

## Technologies Used

* Java
* Object-Oriented Programming (OOP)
* ArrayList
* File Handling (ObjectInputStream / ObjectOutputStream)

## How to Run

1. Compile:
   javac Holiday.java HolidayManager.java

2. Run:
   java HolidayManager

## Notes

* The application automatically creates a data file (`holidays.dat`) when saving data.
* Input validation is implemented to ensure correct user input.
